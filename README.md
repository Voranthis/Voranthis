# 🤖 Telegram Bot — Rotator Ads

**Telegram bot for automatically posting ads in a channel every 30 minutes.**

---

## 📋 Features

- ✅ Sends 4 ads in a cycle every 30 minutes
- ✅ Supports inline buttons for manual scrolling
- ✅ Easy to customize (add your own ads, change timing)
- ✅ Runs 24/7 on a server

---

## 🚀 How to run

1. Clone the repository:
   ```bash
   git clone https://github.com/andrejsulanskij336-pixel/tg_rotator_bot.git

Install dependencies:
pip install python-telegram-bot

Add your token from @BotFather to the TOKEN variable in bot.py.

Run:
python bot.py
